{{ title }}

A comprehensive guide to Angular data binding

1. Interpolation

Purpose: Display component data in the template

Syntax: {{ '{{ expression }}' }}

Message: {{ message }}

Current Date: {{ currentDate | date: "medium" }}

Calculation: 5 + 3 = {{ 5 + 3 }}

String Method: {{ message.toUpperCase() }}

2. Property Binding [property]

Purpose: Set DOM element properties from component data

Syntax: [property]="expression"





Current state: isDisabled = {{ isDisabled }}

3. Attribute Binding [attr.attribute]

Purpose: Set HTML attributes (when no DOM property exists)

Syntax: [attr.attribute-name]="expression"

This cell spans {{ colSpan }} columns
Column 1 Column 2

4. Event Binding (event)

Purpose: Listen to events and execute component methods

Syntax: (event)="handler()"



Last key pressed: {{ lastKeyPressed || "None" }}


Move your mouse here

Mouse Position: X: {{ mouseX }}, Y: {{ mouseY }}

5. Two-Way Binding [(ngModel)]

Purpose: Sync data between component and template (both directions)

Syntax: [(ngModel)]="property"

Note: Requires FormsModule

Hello, {{ userName || "stranger" }}!

Your email: {{ email || "not provided" }}

Selected: {{ selectedColor }}

Agreement status: {{ agreeToTerms ? "Agreed ✓" : "Not agreed" }}

6. Class Binding [class.className]

Purpose: Conditionally add/remove CSS classes

Syntax: [class.class-name]="expression" or [ngClass]="object"

I have conditional classes!


Status Box (using ngClass)

7. Style Binding [style.property]

Purpose: Set inline styles dynamically

Syntax: [style.property]="expression" or [ngStyle]="object"

This text has dynamic styles!

Current font size: {{ fontSize }}px

Summary

Interpolation

{{ '{{ value }}' }}

One-way: Component → Template

Property Binding

[property]="value"

One-way: Component → Template

Event Binding

(event)="handler()"

One-way: Template → Component

Two-Way Binding

[(ngModel)]="value"

Two-way: Component ↔ Template
